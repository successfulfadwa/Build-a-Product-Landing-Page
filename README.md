# Product Landing Page

This is a product landing page that meets the following requirements:

- A header element with a corresponding `id="header"`
- An image within the header element with a corresponding `id="header-img"`
- A nav element within the header element with a corresponding `id="nav-bar"`
- At least three clickable elements inside the nav element, each with the class `nav-link`
- When a `.nav-link` button is clicked, the user is taken to the corresponding section of the landing page
- An embedded product video with `id="video"`
- A form element with a corresponding `id="form"`
- An input field within the form with `id="email"` to enter an email address
- Placeholder text in the `#email` input field to let users know what the field is for
- HTML5 validation in the `#email` input field to confirm that the entered text is an email address
- A submit input within the form with a corresponding `id="submit"`
- When the `#submit` element is clicked, the email is submitted to a static page (mock URL: https://www.freecodecamp.com/email-submit)
- The navbar should always be at the top of the viewport
- At least one media query is used in the landing page
- CSS flexbox is used at least once in the landing page
